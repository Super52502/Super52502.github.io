发表于2022-05-06|更新于2023-02-01

# 一

大家好，第一次做个人博客，看着教程一步一步但还是做了好几天，总是有各种各样的问题和教程中的不一样，其中有各种报错，每一个都要复制到百度去搜索问题的解决办法。刚开始只是配置 git，node.js，下载 hexo，然后又弄主题，然后修改主题（这一步是花费时间最多的），然后又买了一个域名（新用户 1 元买了一年，我看续费好像要花费 69 一年，还是挺贵的）然后又发现评论系统 valine，弄好后又是出现了问题，域名还有备案，刚购买还不能备案，又看到还有搜索系统，等待开发的还有很多。

现在介绍一下我自己吧，目前大二，即将大三，是一个二本带学，没有做过项目，成绩一般，专业信息工程，还是转专业转过来的，对于研究一些花里胡哨的东西比较感兴趣，弄好会有很大成就感。喜欢安卓刷机，Java，js，电脑重装系统，设备维修等，准备把前端深入学习一下，并且把专业课数电、模电、信号系统、高频电路、数字信号处理等学好，本人其实挺喜欢研究数学题，可惜心有余而力不足，等有机会把数学再学一下。

Markdown

因为我们的博文都是用 Markdown 语言写的，所以首先，你需要一个好用的 Markdown 编辑器。其实好用的 Markdown 编辑器一大堆，这里就给大家推荐两个，如果你用的不习惯也可以换其它的。

  • 本地编辑器：Haroopad，非常小众的一款 Markdown 编辑器，左边编辑右边实时预览效果，非常轻便；
  • 在线编辑器：MaHua，也是比较小众的一款 Markdown 编辑器，但效果确实很棒，我的这篇博文就是用 MaHua 写的。

现在你可以打开新建的博文了，然而还不造怎么下手对吧。其实很简单，除了特殊格式，其它的你就当做在 word 里面写文章就行了，具体请看这里的 Markdown 教程：
