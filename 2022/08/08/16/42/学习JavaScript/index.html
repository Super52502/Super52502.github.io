之前自己一点点的学过一些 JavaScript，最近看到 JavaScript 百炼成仙，类修仙的方式把 JavaScript 给学习，感觉挺对我胃口，后期还会学习 Vue 和 springboot 等。

# 一、认识 JavaScript

直接量、变量的声明、数据的类型

数据可以分为两类，

一种是原生数据类型：数字，字符串，布尔值（true/false），null，undefined

一种是对象数据类型，在一个对象里填放数据，一般以键值对的形式存在

访问对象的属性可以用对象 (.) 属性来操作对象属性的值

# 数据绑定

# 处理事件冒泡

在一个对象上触发某类事件（比如单击 onclick 事件），如果此对象定义了此事件的处理程序，那么此事件就会调用这个处理程序，如果没有定义此事件处理程序或事件返回 true, 那么这个事件会向这个对象的的父级传播，从里到外，直至它被处理（父级对象所有同类事件都将被激活），或者到达了对象层次的最顶层，即 document 对象。

vue 提高多种事件修饰符处理事件冒泡
